Example 1: Photos grid in decrease mode

In decrease mode, each row filled by photos until there is no room left. Overflow photos will cause decrese in row's photos widths. 

$(".photos-grid").photosGrid({
    url: "example-data/images.json",
    data_width: "width",
    data_height: "height",
    data_url: "url",
    mode: "decrease",
    margin: 3,
    handleWindowResize: true
});

Example 2: Photos grid in increase mode

In increase mode, each row filled by photos until the next photo has no space left. Then empty space spread equally between row's photos. 

$(".photos-grid").photosGrid({
    url: "example-data/images.json",
    data_width: "width",
    data_height: "height",
    data_url: "url",
    mode: "increase",
    margin: 3,
    handleWindowResize: true
});
